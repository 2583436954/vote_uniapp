项目介绍：
-----------------------------------

<h3 align="center">JEECG BOOT UNIAPP</h3>

实现了与JeecgBoot平台完美对接的移动解决方案！采用技术架构：Uniapp，ColorUI，Vue。

项目目标: 为企业提供一套成熟的APP开发框架，轻松与Jeecg Boot对接，一份代码解决多终端适配（APP、小程序、H5），目前框架已经实现登录机制、用户信息设置、移动首页、菜单等基础功能。


### 功能模块
```
├─框架实现
│  ├─APP开发框架搭建
│  ├─登录对接
│  ├─TOKEN接口机制
│  ├─热更新\覆盖更新
├─基础功能
│  ├─菜单栏目
│  ├─登录页面
│  ├─移动首页
│  ├─个人信息设置
├─消息中心
│  ├─通讯录
│  ├─系统公告
│  ├─消息推送
├─示例代码
│  ├─调用摄像头扫码(扫码)
│  ├─获取地理位置(定位)
├─新增组件
│  ├─页面滚动
│  ├─日历
│  ├─时间选择
│  ├─下拉选择
│  ├─图片上传
├─。。。
```
